# 📄 Manual de Usuario – SISAD Web

## 🧾 1. Introducción

### 🎯 1.1 Objetivo del Manual

Este manual proporciona la guía necesaria para el uso correcto de la plataforma **SISAD Web**, permitiendo al usuario gestionar documentos, realizar consultas y administrar información de manera eficiente.

### 👥 1.2 Audiencia

Dirigido a usuarios del cliente que utilizan la plataforma para:

* Consulta de documentos
* Administración de registros
* Visualización de información

---

## 🌐 2. Descripción General del Sistema

SISAD Web es una aplicación diseñada para:

* Almacenar documentos digitales
* Organizar información mediante índices (metadatos)
* Gestionar documentos vía web
* Clasificar información por fechas, carpetas y estructuras jerárquicas

---


## ⚙️ 3. Requisitos del Sistema

Para el correcto funcionamiento:

* Conexión a Internet
* Navegador web:

  * Google Chrome
  * Mozilla Firefox
  * Microsoft Edge

---

## 📁 6. Gestión Documental

El sistema organiza la información en:

* Gabinetes
* Carpetas
* Subcarpetas

✔ Adaptable según necesidades del cliente

---

## 🖥️ 7. Vista Tradicional

Es la principal vista operativa del sistema donde se gestionan registros y documentos.

---

## 🧩 7.1 Elementos de la Vista Tradicional

### ➕ Botón Crear

Permite crear un nuevo registro ingresando metadata.

---

### ✏️ Botón Editar

Permite modificar la metadata de un registro.

---

### ❌ Botón Eliminar

Permite eliminar registros.

⚠️ **Condición:**
Solo si el registro no tiene archivos asociados.

---

### 📊 Reporte con Filtros Avanzados

Permite generar archivos Excel con:

* Filtros por fechas
* Cantidad de registros
* Inclusión de URLs

✔ Puede tardar según volumen de datos

---

### 🔄 Actualización Masiva

* Permite modificar múltiples registros
* Se realiza mediante archivo Excel

---

### ⬆️ Subida Masiva

#### 📄 Metadata

Pasos:

1. Descargar plantilla
2. Completar datos
3. Subir archivo

⚠️ Formato de fecha: `yyyy-mm-dd`

---

#### 📎 Archivos

* Subir múltiples archivos
* El nombre debe coincidir con el ID SISAD

Campos en plantilla:

* Archivo
* Tipo de archivo
* Nombre (opcional)
* Descripción (opcional)

---

### ⬇️ Descarga Masiva

Opciones:

1. Descargar registros seleccionados
2. Descargar todos los registros
3. Descargar sin filtros

✔ Se puede configurar nombre del archivo con metadatos

---

### 📧 Compartir Archivos

Permite enviar documentos por correo.

Configuraciones:

* Vigencia del enlace
* Tipos de archivo
* Correos en copia (CC)

⚠️ Requiere configuración previa

---

### 📊 Tipos de Archivo como Columnas

Permite visualizar:

* Cantidad de archivos por tipo
* Distribución por registro

✔ Mejora trazabilidad

---

## 🔍 8. Búsqueda

### 🔎 Búsqueda por Metadata

* Ingresar valor en buscador
* Presionar lupa

---

### 📅 Búsqueda por Fecha

* Uso de calendario
* Filtrado por rangos

---

## 📊 9. Índices (Columnas)

Permiten:

* Ordenar información
* Aplicar filtros
* Ajustar visualización

Opciones:

* Fijar columna
* Autoajustar tamaño
* Restablecer columnas

---

## 🧮 10. Filtros

### Tipos:

* Texto
* Numérico
* Fecha (árbol)

✔ Se pueden combinar múltiples filtros

---

### 🔄 Filtros Múltiples

#### Numérico:

* Igual a
* Entre valores
* Mayor/menor

#### Texto:

* Contiene
* Empieza con
* Termina con

---

* Clasificación por tipo
